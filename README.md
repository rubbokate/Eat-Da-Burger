# Eat-Da-Burger
## Description

A burger logger made using MySQL, Node, Express, and Handlebars. Keep track of the burgers you'd like to eat and ones that you have already devoured!

Technologies used
* node.js
* package.json
* MySQL
* Express
* Express Handlebars
* ORM
* MVC design pattern
